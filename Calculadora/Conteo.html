Técnicas de conteo

Permutaciones, permutaciones con r y combinaciones.

Permutaciones

Las permutaciones representan el número de formas en que se pueden ordenar todos los elementos de un conjunto.

  • n: Número total de elementos del conjunto.

Permutaciones con r

Las permutaciones con r elementos toman solo una parte del conjunto y el orden si importa.

  • n: Número total de elementos del conjunto.
  • r: Número de elementos a ordenar o seleccionar.

Combinaciones

Las combinaciones cuentan las formas de seleccionar r elementos de un conjunto de n, sin importar el orden.

  • n: Número total de elementos del conjunto.
  • r: Número de elementos a combinar.
